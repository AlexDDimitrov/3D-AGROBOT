# 3D-AGROBOT
Робот,  който цели да подпомага отглеждането и проследяването на здравословното състояние на селскостопански насаждения.


# Kодове и тяхното значение за разработката:
- 0 - успех с операцията, заявката
- 101 - невалидно име
- 102 - невалидна парола
- 103 - невалидно име
- 104 - този имейл се използва от друг потребител
- 105 - грешна парола или имейл
- 201 - токена е изтекъл или невалиден
- 202 - липсват полета
- 203 - не съществува тази градина или не е собственост на потребителя
